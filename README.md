Description

This project contains two parts:

1. File Encryption & Decryption
2. Expression Parser and Evaluator


##  Part 1: Encryption

### Rules:
* Lowercase (a–m): shift forward by (shift1 × shift2)
* Lowercase (n–z): shift backward by (shift1 + shift2)
* Uppercase (A–M): shift backward by shift1
* Uppercase (N–Z): shift forward by (shift2²)
* Other characters remain unchanged

### Files:

* raw_text.txt
* encrypted_text.txt
* decrypted_text.txt

## Part 2: Expression Parser

### Features:

* Tokenization
* Parse Tree generation
* Expression evaluation
* Error handling:

  * Invalid operators
  * Division by zero
